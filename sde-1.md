# Training Modules – 1 Year Developer Expectations

> This document organizes training expectations into clear learning modules.
> Scope is aligned with **1-year backend / platform developer expectations**.


---

## Module 1: Java 
**Topics**  
-   Collections
-   Lambda
-   Stream api
-   Generics
-   String, String builder, String buffer

**Description**  
Covers Java fundamentals required for enterprise backend development. Focus on memory behavior, performance implications, immutability, and modern Java functional style using streams and lambdas.

**Resources**  
- Scaler – Collections: https://www.scaler.com/topics/java/collections-in-java/
- Scaler - java8: https://www.scaler.com/topics/java-8-features/
- Java Internals & Performance: https://www.freecodecamp.org/news/understanding-java-internals-speed-and-performance/
