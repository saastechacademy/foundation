## AppSec - Application Security Engineer

This folder will contain all the details that can help to become an Application Security Engineer.

An Application Security Engineer (AppSec) focuses on ensuring that software applications are secure throughout their lifecycle—from design to deployment. They identify and fix vulnerabilities such as injection flaws, authentication issues, and misconfigurations by performing code reviews and security testing. AppSec engineers work closely with developers to embed secure coding practices and follow standards like OWASP guidelines. They also conduct threat modeling, secure APIs, and help integrate security into the development process (Shift-Left approach). Their goal is to prevent security issues before they reach production and reduce overall risk in applications.
